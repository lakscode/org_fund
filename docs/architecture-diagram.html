Request Flow

sequenceDiagram participant B as Browser (React) participant A as Axios Interceptor participant S as HTTP Server :8000 participant R as Router (regex) participant RT as Route Handler participant M as Model participant DB as MongoDB B->>A: API call (e.g. GET /api/funds/:id/balancesheet) A->>A: Attach Bearer token from localStorage A->>S: HTTP Request + Authorization header S->>S: CORS headers S->>R: Parse path + method R->>R: Match regex pattern R->>R: _auth_or_401() - validate JWT R->>RT: Call route handler(params, user) RT->>M: Call model function M->>DB: PyMongo query DB-->>M: Documents M-->>RT: Processed data RT-->>R: (status_code, response_dict) R-->>S: MongoEncoder JSON serialize S-->>A: HTTP Response A->>A: Check status (401 → logout) A-->>B: Response data

Authentication Flow

sequenceDiagram participant U as User participant UI as React App participant API as Backend :8000 participant DB as MongoDB U->>UI: Enter email + password UI->>API: POST /api/auth/login API->>DB: Find user by email DB-->>API: User document API->>API: Verify password (Passlib bcrypt) API->>API: Create JWT token (PyJWT) API-->>UI: {access_token} UI->>UI: Store token in localStorage UI->>API: GET /api/auth/me (Bearer token) API->>API: Decode JWT → user_id API->>DB: Find user + orgs + roles DB-->>API: User + org data API-->>UI: {id, name, email, orgs[{id, name, role}]} UI->>UI: Set AuthContext (user, currentOrg) UI->>UI: Render Layout with role-based sidebar
